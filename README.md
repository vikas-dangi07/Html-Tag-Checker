# HTML Tag Checker

## 📌 Overview
HTML Tag Checker is a web-based tool that analyzes HTML code and detects structural errors such as missing, mismatched, or improperly nested tags. It helps developers and students debug HTML efficiently.

## 🚀 Features
- Detects missing HTML tags
- Identifies mismatched tags
- Validates nested structure
- Simple UI for quick checking
- Fast processing using Flask backend

## 🛠 Technologies Used
- Python (Flask)
- HTML
- CSS
- JavaScript

## 📂 Project Files
app.py – Flask backend  
index.html – User interface  
styles.css – Styling  
main.js – Validation logic  

## ▶ How to Run

Step 1:
Clone repository

Step 2:
Install Flask

pip install flask

Step 3:
Run application

python app.py

Step 4:
Open browser

http://127.0.0.1:5000
